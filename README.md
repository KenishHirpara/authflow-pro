# 🚀 AuthFlow Pro API

## Features
- JWT Authentication (Register & Login)
- Protected Routes
- Payment Mock API
- Image Upload (Cloudinary)

## Tech Stack
Node.js, Express, MongoDB, JWT, Multer, Cloudinary

## API Endpoints
- POST /api/auth/register
- POST /api/auth/login
- GET /api/auth/protected
- POST /api/payment/pay
- POST /api/upload/upload
